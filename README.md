#  Personal Portfolio Website

A modern and professional personal portfolio website built using **React**, **Tailwind CSS**, and **JavaScript**.  
This portfolio showcases my skills, projects, experience, and contact information with a clean UI, smooth animations, and responsive design.

---

##  Features

-  Dark & Light Theme Toggle
-  Fully Responsive Design
-  Modern Professional UI/UX
-  Skills & Technologies Section
-  Projects Showcase with Details
-  Contact Page / Contact Form
-  Smooth Scrolling & Animations
-  Clean and Minimal Layout
-  Fast Performance

---

##  Built With

- **React.js**
- **JavaScript (ES6+)**
- **Tailwind CSS**
- **HTML5**
- **CSS3**
